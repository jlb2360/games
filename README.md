# Games

## Summary
This is a folder that holds games that can be played on the terminal. Below is a list of games.

## Instructions
To build a game follow these directions
1. cd <gamee>
2. mkdir build
3. cmake ../
4. cmake --build .
5. ./main (this starts the game)
